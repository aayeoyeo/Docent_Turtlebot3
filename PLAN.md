# 🤖 자율주행 AI 도슨트 로봇 시스템 계획서 — 4조

## 1. 프로젝트 개요
본 프로젝트는 **ROS 2(Humble)** 환경의 터틀봇3(Burger)와 스마트폰 센서 인프라를 웹소켓으로 융합하여, 전시관 내에서 관람객을 안전하게 안내하고 작품을 다국어로 해설하는 **'자율주행 AI 도슨트 로봇 시스템'** 구축을 목표로 합니다. 로봇은 주행 중 벽면에 부착된 QR 코드를 통해 자신의 위치와 전시물을 식별(`MOVE` ➔ `EXPLAIN`)하고, JSON 코스 데이터 기반의 정밀한 TTS 해설을 제공합니다. 해설이 종료되면 디바이스 카메라와 경량 AI 추론 엔진(`tracking.js`)을 활용해 관람객의 얼굴을 대면 인식(`CHECK`)한 뒤 다음 목적지로 자율 이동하며, 비상시 관제소 웹 인터페이스를 통해 즉각적인 수동 원격 제어(`MANUAL`) 및 실시간 Base64 CCTV 스트리밍 모니터링이 가능한 고안전성 지능형 상태 머신 제어 시스템을 완성합니다.

## 2. 기능 요구사항 (MoSCoW)
- **Must (필수 구현)**
  - 스마트폰 카메라를 통한 전시물 QR 코드(E1, E2 등) 정밀 인식 및 지정 위치 정지 기능
  - JSON 코스 데이터 스크립트 기반의 Web Speech API 활용 상황 인지형 TTS 해설 기능
  - 해설 종료 후 `tracking.js` 기반의 관람객 안면 인식 및 차기 코스 연계 상태 전이 기능
  - 관제소 웹 페이지와의 대화형 통신 인터페이스 및 실시간 상태(IDLE, MOVE, EXPLAIN, CHECK) 모니터링
- **Should (우선 구현)**
  - 비상 상황 대응을 위한 관제탑 강제 개입 조이패드 원격 수동 조종 기능 (`MANUAL` 모드)
  - 로봇의 전방 시야 확보를 위한 웹 화면 기반 실시간 CCTV 스트리밍 기능 (250ms 주기 Base64 인코딩 전송)
  - QR 미인식 타임아웃(30초) 및 관람객 부재 타임아웃(20초) 발생 시 예외 처리 및 자동 정지/강제 이동 로직
- **Could (권장 구현)**
  - 안면 인식의 정확도 향상 및 오판정 방지를 위한 프레임 누적 디바운스(Debounce) 필터 로직
  - 현재 주행 경로 및 진행률(%)의 관제소 대시보드 시각화 인디케이터 UI 구현
- **Won't (추후 구현)**
  - 다중 로봇 운영을 위한 다자간 자율 협업 및 동적 장애물 회피 알고리즘 (SLAM/Nav2 연동 복합 주행)

## 3. 시스템 아키텍처
본 시스템은 하드웨어 제어부(라즈베리파이 4), 로봇 시각/청각 서브시스템(스마트폰 웹앱), 중앙 원격 관제탑(우분투 노트북 웹 서버)의 3단 토폴로지로 구성되며, `rosbridge_suite` 기반의 WebSocket 프로토콜을 통해 실시간 비동기식 메시지 교환을 수행합니다.


```

[스마트폰 뷰어 (robot.html)] ──(WebSockets: 9090)──┐

* QR 코드 및 안면 인식 추론 (tracking.js)      │
* 실시간 비디오 프레임 전송 / TTS 출력         │
▼
[원격 관제탑 (admin.html)] ───(WebSockets: 9090)───▶ [노트북 / 라즈베리파이 (ROS 2 Humble)]
* JSON 투어 코스 주입 및 강제 Phase 제어             - docent_node (중앙 제어 상태머신)
* 실시간 CCTV 모니터링 및 D-Pad 수동 조종             - turtlebot3_bringup (/cmd_vel 구동)

```
*(상세 구조도는 아키텍처 다이어그램 파일 `images/architecture.png` 참조)*

## 4. 사용 기술
- **Hardware:** TurtleBot3 Burger, Raspberry Pi 4 (4GB), 안드로이드/iOS 스마트폰, Ubuntu 22.04 LTS 탑재 노트북
- **Software / Framework:** ROS 2 Humble, Python 3.10, rclpy, rosbridge_suite
- **Web & AI Interface:** HTML5 Canvas, JavaScript (ES6), roslibjs, html5-qrcode.js, tracking.js (Face Detection), Web Speech API (TTS)
